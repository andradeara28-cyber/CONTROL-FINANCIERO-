El "Truco Secreto" de GitHub (Mi favorita) GitHub tiene un editor oculto que es idéntico a **Visual Studio Code** (el programa que usan los programadores profesionales) pero corre 100% dentro de tu navegador. * **Cómo usarlo:** Entra a tu repositorio en GitHub desde tu computadora y simplemente **presiona la tecla del punto `.**` en tu teclado. * **Qué pasa:** Magia. Tu pestaña de GitHub se transformará en un editor de código completo en segundos. Ahí puedes crear tus archivos `index.html`, `styles.css`, organizar tus carpetas y escribir todo cómodamente.
